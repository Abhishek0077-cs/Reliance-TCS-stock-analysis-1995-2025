# Reliance-TCS-stock-analysis-1995-2025
A data analysis project exploring 30 years of stock price trends of Reliance Industries and TCS. The study includes data cleaning, exploratory data analysis, and visualizations to compare long-term growth, volatility, and market performance.


## Overview
This project analyzes 30 years of stock price trends for Reliance Industries and TCS.

## Objectives
- Analyze long-term stock trends
- Compare performance between the two companies
- Visualize price movement and volatility

## Dataset
Historical stock price data for Reliance and TCS.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly


## Insights 

1• TCS shows consistent long-term growth.
2• Reliance shows higher volatility.
3• Both stocks have strong long-term returns.

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
